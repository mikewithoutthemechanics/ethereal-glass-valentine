# Ethereal Glass Valentine — Quick Start

1. **Customize** `config.json` with your names, messages, and captions
2. **Drop photos** into `assets/` as `candice1.jpg`, `candice2.jpg`, etc.
3. **Test** — open `index.html` locally (double-click or `python3 -m http.server 8000`)
4. **Deploy** — drag folder to https://app.netlify.com/drop OR use `netlify deploy --prod`

That's it. Share the link with Candice 💖
